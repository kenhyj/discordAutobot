# Autobot - Discord bot

## To run changes to Autobot,

For the first time:

```bash
$ npm install
```

after that:

```bash
$ node bot.js
```

#### Text Commands

- _!babyshark_ : answers with "Baby shark, doo doo doo doo doo doo~ Baby shark, doo doo doo doo doo doo \n !play baby shark"
- _!fliptable_ : answers with "┛◉Д◉)┛ 彡 ┻━┻"
